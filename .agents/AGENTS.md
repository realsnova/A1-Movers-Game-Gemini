# 專案開發規則 (A1 Movers 單字冒險)

這是一個為台灣國小三年級設計的英文單字學習遊戲專案。當你（Antigravity）在這個專案中協助開發時，請嚴格遵守以下規則：

## 1. 核心技術棧與架構
- **純原生開發**：只使用 HTML, Vanilla CSS, Vanilla JavaScript。絕對**不要**引入 React, Vue, Tailwind 等框架。
- **單一檔案交付**：最終交付物必須是一個名為 `單字冒險_Gemini.html` 的單一檔案。
- **建置流程**：原始碼放在 `src/` 目錄下（包含 `index.html`, `main.js`, `style.css` 等）。任何修改都必須在 `src/` 中進行，修改完成後，**必須執行 `python build.py`** 來將所有的 JS、CSS 與 JSON 資料內嵌打包成最終的單一 HTML 檔。

## 2. 遊戲系統與資料
- 遊戲資料位於 `wordlist.json` (題目) 與 `monsters.json` (圖鑑與進化鏈)。
- 遊戲進度狀態存放在 `localStorage` (Key: `A1MoversState`)。
- 遊戲包含學習模式 (SRS系統) 與戰鬥模式 (打怪與道館倒數計時)。

## 3. 開發風格與原則
- **極致防呆**：遊戲受眾為 8-9 歲兒童，任何按鈕與過場都必須處理好狀態鎖定，避免狂點造成的 Bug。
- **視覺回饋**：重視 Game feel（如受擊震動、血條變色、音效提示）。
- **保留繁體中文**：所有介面與註解請使用繁體中文（zh-TW）。

## 4. 歷史進度與重要資訊
- 專案已經完成 Phase 1 ~ Phase 7 的深度打磨（包含退格鍵、提示藥水、圖鑑九宮格、填空題防壞題等）。
- 嚴禁破壞現有的 `main.js` 戰鬥與學習迴圈。
